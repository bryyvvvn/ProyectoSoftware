# ProyectoSoftware

## Requisitos

- Node.js 18 o superior
- npm

## Ejecución del servidor

1. Abrir una terminal y ubicarse en la carpeta `server`.
2. Instalar dependencias con:
   ```bash
   npm install
   ```
3. Iniciar el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. El servidor quedará disponible en `http://localhost:3000`.

## Ejecución de la aplicación

1. En otra terminal, ir a la carpeta `App`.
2. Instalar dependencias con:
   ```bash
   npm install
   ```
3. Iniciar la aplicación React:
   ```bash
   npm run dev
   ```
4. Vite mostrará en consola la URL (por defecto `http://localhost:5173`). Abrirla en el navegador.
